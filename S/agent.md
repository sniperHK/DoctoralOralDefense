# S — Agent Rules（AI 協作規範）

## 讀取順序（避免迷路）

1. `ICGLPS.md`
2. `C/ACTIVE.md`
3. `P/TODO.md`
4. 需要證據再去 `C/`（官方文件、考古題、教材）

## 產出放置規則（One change, one place）

- 任何「可交付」的整理（答題模板、題型整理、一頁式重點、模擬考檢討）→ `G/`
- 待辦/排程/下一步 → `P/`
- 仍有效的規定與資料入口 → `C/`
- 完成項摘要與版本留痕 → `L/CHANGELOG.md`

## 答題輸出格式（偏考試導向）

- 先寫：定義/核心主張（1–3 句）
- 再寫：架構（用條列對應得分點）
- 補強：例子/情境化應用（至少 1 個）
- 最後（可選）：限制/反思/替代解釋（視題型與配分）

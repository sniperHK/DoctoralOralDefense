# C — ACTIVE（現行規定與範圍摘要）

## 考試結構（以官方文件為準）

- 兩科同日考：研究法（3 小時；9:00–12:00）＋健康行為與社區科學（4 小時；13:00–17:00）
- 配分與題型（摘要）：
  - 研究法：必選題 70%＋選考題 30%（四選二）
  - 健康行為與社區科學：基礎題 60%＋進階題 40%（通常選一領域作答）
- 官方說明與書單：`C/博班資格考考古題/健管所博士班行社組資格考_研究法與專業考科說明及書單(20220928所務會議通過，自 112 學年度起實施).pdf`

## 研究法（常見題型線索：以考古題整理）

- 概念題：謬誤（如 atomistic fallacy）、分析單位（unit of analysis）
- 研究設計與選擇理由：橫斷/世代/實驗/類實驗/趨勢研究等
- 抽樣：定義、優缺點、執行步驟（常出大題）
- 系統性文獻回顧：步驟＋每一步的理由
- 調查方法比較：不同調查方式、優缺點、選擇理由、如何找舊資料/歷史數據
- PICO 設計與 Outcome 測量
- 質性研究信效度：質性對信度/效度的理解、與量化差異、如何提升研究品質

（參考例題：`C/博班資格考考古題/資格考考古題-研究法(行社組)/112-2 健管所博士班資格考_研究法(行社組).pdf`）

## 健康行為與社區科學（常見題型線索：以考古題整理）

- 污名與健康倡議：污名定義/面向/例子；倡議時避免污名傷害
- 社區工作：ABCD（資產導向）與需求導向差異
- 行為改變理論：TTM（階段、螺旋模式、決策平衡、限制）
- 整合模型：COM-B 與 behavior change wheel（介入功能與策略設計）
- 健康不平等：difference/disparity/inequality/inequity；Gini index、concentration index

（參考例題：`C/博班資格考考古題/資格考考古題-研究法(行社組)/112-2 健管所博士班資格考_健康行為與社區科學.pdf`）

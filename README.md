# DoctoralOralDefense / Qualifying Exam Workspace (ICGLPS)

這個 repo 是一個用 **ICGLPS**（I/C/G/L/P/S）管理的博士班里程碑準備專案（目前以「行社組資格考：研究法」為主要範例）。

## 你可以用它做什麼

- 用 ICGLPS 結構管理：規定/脈絡（`C/`）、產出（`G/`）、待辦（`P/`）、留痕（`L/`）、協作規範（`S/`）
- 透過瀏覽器介面練考古題：計時作答 → LLM 評分回饋 → 指向書單主題校正

## 快速入口

- 專案控制檔：`ICGLPS.md`
- 現行範圍摘要：`C/ACTIVE.md`
- 待辦清單：`P/TODO.md`
- 研究法樣本筆記（含官方書單對照）：`G/模擬考/112-2_研究法_重點筆記.md`
- 互動練題工具：`G/llm-exam-game/README.md`

## 版控策略（重要）

本 repo 預設 **不會** 把下列內容推到 GitHub（已在 `.gitignore` 排除）：

- 教材/章節 PDF、考古題 PDF、校內表單、截圖、zip 備份

這些內容請放在 `C/` 或 `L/archives/` 對應資料夾內做本機管理即可（各資料夾內有 `README.md` 說明）。
